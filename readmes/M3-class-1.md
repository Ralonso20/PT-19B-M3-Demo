### Inicio del proyecto

```bash
npm init -y
```

#### Instalar typescript como dependencia dev

```bash
npm i -D typescript
```

#### Iniciar Typescript

```bash
npm tsc --init
```

#### Instalacion Prettier y Eslint

```bash
npm i -D eslint prettier typescript-eslint eslint-config-prettier eslint-plugin-prettier
```

#### Iniciar eslint

```bash
npm eslint --init
```
